# Main Store

Proyecto Vue 3 para gestionar múltiples tiendas.

## Scripts

- `npm run serve` — Ejecuta el servidor de desarrollo.
- `npm run build` — Compila para producción.
- `npm run lint` — Lint del código.
- `npm run deploy` — Despliega a GitHub Pages.

## Demo

[Ver tienda en producción](https://jorgelsc-dev.github.io/main-store/)

## Estructura

- `src/` — Código fuente.
- `docs/` — Archivos estáticos para GitHub Pages.

## Dependencias principales

- Vue 3
- Vue Router
- Vuex
- Vuetify
